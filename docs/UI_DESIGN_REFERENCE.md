### Color Scheme

- **Primary Background**: Light teal gradient (`from-teal-50 via-[#E6FFFA] to-teal-100`)
- **Cards**: White with 95% opacity (`bg-white/95`)
- **Borders**: Light teal (`border-teal-200`)
- **Text Primary**: Dark teal (`text-teal-900`)
- **Text Secondary**: Medium teal (`text-teal-700`)
- **High Risk Badges**: Red (`bg-red-100 text-red-800 border-red-200`)
- **Medium Risk Badges**: Orange (`bg-orange-100 text-orange-800 border-orange-200`)
- **Low Risk Badges**: Green (`bg-green-100 text-green-800 border-green-200`)

   - Orange: 40-69 (Medium)

### Interactive Elements

- **Table Rows**: Hover effect (background color change) to indicate clickability
- **Filters**: Dropdown menus with search capability
- **Search Input**: Shows search history on focus
- **Summary Cards**: Clickable (future: navigate to filtered view)

### Responsive Design

- **Desktop**: Full layout with all columns visible
- **Tablet**: Summary cards stack vertically, table scrolls horizontally
- **Mobile**: Single column layout, filters stack vertically
